Support

  • Chat

    To get quick access to support, feel free to use our dedicated workspace on Slack.

  • Q&A

    To ask questions (or search existing ones), use the corresponding "Stack Overflow" lable.

  • Task management

    A task management system where you can report bugs, monitor the progress of issues, and suggest new features can be found in the corresponding GitHub repositories.

  • Blog

    For more articles and other information about the platform, please follow our blog on Habr.com.

  • News

    You can keep track of our news and updates by following us on Twitter.

  • Source code

    The source code for the platform and IDEA plugin can be found in the corresponding GitHub repositories.

  • Build system

    You can monitor auto-build processes using the corresponding Jenkins Web UI.

  • Email

    For all other questions, please email us at info@lsfusion.org.
